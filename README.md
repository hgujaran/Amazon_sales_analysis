##  Overview
This project analyzes Amazon product sales data to uncover insights related to
category performance, pricing behavior, discount impact, revenue contribution,
and customer engagement. The analysis was performed using Microsoft Excel with
a focus on data cleaning, KPI modeling, and interactive dashboards.

---

## Tools used
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Excel Functions & Formulas  
- Data Cleaning  
- KPI   
- Data Visualization  

---

##  Dataset Description
The dataset contains Amazon product-level sales information, including:
- Product category
- Price and discount percentage
- Revenue indicators
- Customer ratings 

The dataset used for analysis is available in the data/ folder.

---

##  Analysis Performed
- Category wise revenue contribution analysis  
- Impact of discounts on customer engagement  
- Price range analysis and revenue proxy evaluation  
- Identification of high-performing and underperforming categories  
- KPI-driven dashboard creation using Pivot Tables

##  Key Insights
- Electronics accessories and home categories showed the highest revenue contribution  
- Higher discount percentages correlated with increased customer rating activity,
  indicating stronger customer engagement  
- Certain categories exhibited price inflation without a proportional increase in ratings,
  suggesting weaker perceived value
