This project analyzes Amazon product data to understand category performance, pricing patterns, discount behavior, and customer engagement using Microsoft Excel. The work includes data cleaning, KPI backend modeling, and dashboard creation using Pivot Tables and Charts.

Tools used

Microsoft Excel

Pivot Tables & Pivot Charts

KPI Backend Modeling

Data Cleaning & Transformation

Excel Functions & Formulas

Data Visualization 

Analytical Insights

Electronics accessories and home categories showed the highest revenue contribution.

Higher discount percentages correlated with increased customer rating activity, indicating higher engagement.

Certain categories showed high price inflation without corresponding increases in ratings, suggesting weaker perceived value.
